# basic modules cheatsheet

## Datetime Module

Used for working with dates and times.

| Function/Attribute                  | Purpose                                          | Expense Tracker Use Case                                                    |
| :---------------------------------- | :----------------------------------------------- | :-------------------------------------------------------------------------- |
| `date.today()`                      | Gets the current date (as a `date` object).      | Used to stamp a new expense with the current day.                           |
| `.year`, `.month`                   | Attributes of a `date` object.                   | Used in `summarize_monthly` to filter expenses for the current period.      |
| `.strftime(format)`                 | Formats a `date` object into a specified string. | Used in `_write_to_csv` to ensure the CSV date is `YYYY-MM-DD`.             |
| `datetime.strptime(string, format)` | Parses a string into a `datetime` object.        | Used in `_read_from_csv` to convert the CSV string back into a date object. |
