## ‚ö†Ô∏è Data Integrity Principle (CRITICAL LESSON)

The most important takeaway was ensuring **data type consistency** between memory and file:

1. **Read:** `String` (from CSV) $\rightarrow$ `date` object & `float` (in Python list).
2. **Process:** Use native types for all math and logic (e.g., `expense['Amount'] + new_amount`).
3. **Write:** `date` object & `float` (in Python list) $\rightarrow$ `String` (to CSV via `.strftime()`).
